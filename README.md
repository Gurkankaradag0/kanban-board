## Setup

-   Clone the repository:
    ```bash
    git clone https://github.com/Gurkankaradag0/kanban-board.git
    cd kanban-board-api
    ```
-   Install dependencies:

    ```python
    npm install or yarn install or pnpm install
    ```

## Run Project

```python
npm dev or yarn dev or pnpm dev
```
